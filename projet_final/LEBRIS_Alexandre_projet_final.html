Vendée Globe 2020-2021
"Des formules 1 dans les dunes du Paris-Dakar" ?

L'édition 2020 - 2021 du Vendée Globe a été une des plus haletantes jamais courues, avec 8 participants qui se sont tenus à moins de 24h d'écart pour près de 80 jours de course. Toutefois, malgré cette compétition féroce entre skippers, le record d'Armel Le Cléac'h, établi à 74 jours lors de la précédente édition, n'a jamais semblé pouvoir être battu, alors même que de nombreux voiliers de dernière génération, équipés de foils, étaient alignés.

Plus surprenant encore, les voiliers classiques, mis à l'eau généralement il y a un plus grand nombre d'années, se sont invités dans le sprint final, faisant parfois jeu égal avec leurs homologues modernes. Ce constat, qui pourrait passer pour une anomalie à première vue, a fait dire à Jean Le Cam, un de ces skippers "à l'ancienne", que l'on avait envoyé "des formules 1 dans les dunes du Paris-Dakar".

Regardons ensemble, sur la base des données proposées par le site officiel du Vendée Globe, si cette déclaration peut se révéler fondée à travers les chiffres de course.

Acquisition des données¶

Dans cette partie, nous allons télécharger les données utiles à notre analyse, à savoir :

  • Les données de course, publiées toutes les 4 heures pour l'ensemble des participants et constituées de certaines mesures instantanées (rang, longitude, latitude, distance au leader ou restante avant le finish) et de certaines aggrégées (vitesse, cap, distance enregistrés lors des dernières 24h, 4h ou 30 min). Celles-ci sont compilées au sein de fichiers Excel qu'il est possible de télécharger.
  • Les données techniques des bateaux, disponibles sur le site officiel du Vendée Globe, qu'il faut scrapper pour pouvoir les obtenir et les utiliser.

def download_boat_data():
    # Chargement des données sur les bateaux
    page = requests.get(PATH_RANK_BOAT)
    soup = BeautifulSoup(page.content, 'html.parser')
    boat_list = soup.select('div.boats-list__popup')
    
    # Extraction des informations au sein d'un Dataframe
    data = []
    for boat in boat_list:
        specs = boat.select('li')
        boat_data = {
            key: value for key, value in
            [spec.contents[0].split(" : ")[:2] for spec in specs]
        }
        data.append(boat_data)
        
    return pd.DataFrame(data=data)

Préparation des données¶

Une fois les données acquises, il est temps de nettoyer les données pour pouvoir les utiliser en vue des analyses à mener. De manière générale, ces opérations incluent :

  • Une conversion des features en type de données utilisables par les bibliothèques utilisées : les positions géographiques, les dates/temps, les durées, les distances, les vitesses, les caractéristiques des bateaux, etc.
  • Une normalisation des colonnes, qui serviront de clé lors du croisement des données (dans notre cas, ce sera le numéro de voile).
  • Calcul de nouvelles features comme l'âge des bateaux, ou encore la zone géographique traversée.
  • A noter également une anomalie sur les fichiers Excel téléchargés qui empêche le chargement par pandas, et qu'il faut préalablement corriger.

# Fonction pour "réparer" les fichiers Excel téléchargés (présence 
# d'un mot-clé qui empêche l'ouverture du fichier par openpyxl et donc pandas)
def fix_excel(file_name, name_filter, change):
    tempdir = tempfile.mkdtemp()
    
    try:
        tempname = os.path.join(tempdir, 'new.zip')
        with ZipFile(file_name, 'r') as r, ZipFile(tempname, 'w') as w:
            for item in r.infolist():
                data = r.read(item.filename)           
                data = change(data)
                w.writestr(item, data)
        shutil.move(tempname, file_name)
    finally:
        shutil.rmtree(tempdir)

# Fonction pour convertir les degrés, minutes, secondes en dégrés décimaux
def dms2dd(dms):
    degree, minute, second = re.findall("(\d+)", dms)
    dd = int(degree) + int(minute) / 60 + int(second) / 3600
    
    if dms[-1] in 'SW':
        return -dd
    return dd

# Fonction pour convertir les écarts en temps en timedelta
def duration2timedelta(item):
    if item is np.nan:
        return item
    
    figures = re.findall("(\d+)", item)
    
    if len(figures) == 4:
        days, hours, minutes, seconds = figures
    elif len(figures) == 3:
        hours, minutes, seconds = figures
        days = 0
    else:
        minutes, seconds = figures
        days= 0
        hours = 0
        
    return datetime.timedelta(days=int(days), seconds=int(seconds),
                              minutes=int(minutes), hours=int(hours))

# Fonction pour charger et nettoyer les données d'un fichier Excel
def import_clean_excel(filename):   
    columns = ["rang", "voile", "nom_skipper", "date_heure", "lat", "long",
               "cap_30min", "vit_30min", "vmg_30min", "distance_30min",
               "cap_prec", "vit_prec", "vmg_prec", "distance_prec",
               "cap_24h", "vit_24h", "vmg_24h", "distance_24h",
               "dtf", "dtl"]

    # Import du fichier Excel dans un DataFrame
    df = pd.read_excel(filename, skiprows=4, nrows=33, usecols="B:U")
    df.columns = columns

    # Extraction des lignes pour les coureurs toujours en course
    df = df[(~df.nom_skipper.isna()) & (~df.date_heure.isna())]
    
    if len(df) == 0:
        return df

    # Traitement de la colonne rang
    df.rang = pd.to_numeric(df.rang)

    # Traitement de la colonne voile
    df.voile = df.voile.apply(lambda line: re.findall("(\d+)", line)[0])

    # Traitement de la colonne nom_skipper
    df.nom_skipper = (df.nom_skipper
                         .apply(lambda line: re.match("^(.*)\\n", line).group(1))
                         .str.lower())

    # Traitement de la colonne heure
    date = filename[17:25]
    heure = filename[26:32]
    df.date_heure = df.date_heure.apply(lambda line: date + " " + heure)
    df.date_heure = pd.to_datetime(df.date_heure, format="%Y%m%d %H%M%S")
    df.date_heure = df.date_heure - datetime.timedelta(minutes=30)

    # Traitement des colonnes latitude et longitude
    df["geo"] = df.apply(lambda line: (dms2dd(line.lat), dms2dd(line.long)), axis=1)

    # Traitement des colonnes heading
    columns = ["cap_30min", "cap_24h", "cap_prec"]
    df[columns] = df[columns].applymap(lambda item: float(item[:-1]))

    # Traitement des colonnes speed
    columns = ["vit_30min", "vit_24h", "vit_prec"]
    df[columns] = df[columns].applymap(lambda item: float(item[:-4]))

    # Traitement des colonnes vmg
    columns = ["vmg_30min", "vmg_24h", "vmg_prec"]
    df[columns] = df[columns].applymap(lambda item: float(item[:-4]))

    # Traitement des colonnes distance
    columns = ["distance_30min", "distance_24h", "distance_prec"]
    df[columns] = df[columns].applymap(lambda item: float(item[:-3]))

    # Traitement des colonnes dtf et dtl
    columns = ["dtf", "dtl"]
    df[columns] = df[columns].applymap(lambda item: float(item[:-3]))
    
    return df

# Aggrégation des données dispersées sur les différents fichiers Excel au sein d'un unique Dataframe,
# en utilisant la fonction définie ci-dessus
def generate_df_course(filename_list):   
    df_list = []
    
    # Réparation, chargement et nettoyage de chaque fichier Excel
    for filename in filename_list:
        fix_excel(filename, 
                  name_filter='xl/styles.xml', # L'anomalie se trouve dans le fichier xml styles
                  change=lambda d: re.sub(b'xxid="\d*"', b"", d)) # On enlève le mot-clé xxid
        df_list.append(import_clean_excel(filename))
    
    # Création du Dataframe final
    df_course = pd.concat(df_list)
    
    # Suppression des dupliqués
    df_course = df_course.drop_duplicates()
    
    # Suppression des colonnes inutiles et reset
    df_course = df_course.drop(columns=['lat', 'long'])
    df_course = df_course.reset_index().drop(columns='index')
    
    return df_course

def generate_df_coureur(filename_list):   
    # Import du dernier fichier Excel en date (avec tous les résultats finaux)
    df_coureur = pd.read_excel(sorted(filename_list)[-1],
                               skiprows=4, nrows=33, usecols="B:D, H:I, N, P:U")
    columns = ["status", "nat_sail", "skipper_crew", "date_arrivee", "temps_course",
               "ecart_premier", "ecart_prec", "vit_ortho", "distance_ortho",
               "arrow", "vit_fond", "distance_fond"]
    df_coureur.columns = columns

    # Création de la colonne final_ranking
    df_coureur['final_ranking'] = df_coureur.status.apply(lambda item: re.findall("(\d*)", item)[0])
    df_coureur.final_ranking = pd.to_numeric(df_coureur.final_ranking)

    # Modification de la colonne status
    df_coureur.status = df_coureur.status.apply(lambda item: "ARV" if "ARV" in item else "RET")

    # Création des colonnes skipper et crew
    df_coureur['nom_skipper'] = (df_coureur.skipper_crew
                                 .apply(lambda item: re.findall("^(.*)\\n", item)[0]))
    df_coureur['nom_bateau'] = (df_coureur.skipper_crew
                              .apply(lambda item: re.findall("\\n(.*)", item)[0]))

    # Création des colonnes nat et sail
    df_coureur['nat'] = df_coureur.nat_sail.apply(lambda item: re.findall("\\n(\w*)", item)[0])
    df_coureur['voile'] = df_coureur.nat_sail.apply(lambda item: re.findall("(\d*)$", item)[0])

    # Nettoyage de la colonne date_arrivee
    df_coureur.date_arrivee = pd.to_datetime(df_coureur.date_arrivee, format="%d/%m/%Y %H:%M:%S FR")

    # Nettoyage de la colonne temps_course
    df_coureur.temps_course = df_coureur.temps_course.apply(lambda item: item.split("\n")[0]
                                                      if isinstance(item, str) else item)
    df_coureur.temps_course = df_coureur.temps_course.apply(duration2timedelta)

    # Création de la colonne compensation
    compensation = {
        "17": datetime.timedelta(hours=-10, minutes=-15),
        "01": datetime.timedelta(hours=-16, minutes=-15),
        "10": datetime.timedelta(hours=-6),
        "02": datetime.timedelta(hours=2)
    }
    df_coureur['compensation'] = df_coureur['voile'].map(compensation)

    # Conversion des colonnes ecart_premier et ecart_prec
    df_coureur.ecart_premier = df_coureur.ecart_premier.apply(duration2timedelta)
    df_coureur.ecart_prec = df_coureur.ecart_prec.apply(duration2timedelta)

    # Conversion des colonnes speed
    columns = ["vit_ortho", "vit_fond"]
    df_coureur[columns] = df_coureur[columns].applymap(lambda item: float(item[:-4])
                                                       if item is not np.nan else item)

    # Conversion des colonnes distance
    columns = ["distance_ortho", "distance_fond"]
    df_coureur[columns] = df_coureur[columns].applymap(lambda item: float(item[:-3])
                                                       if item is not np.nan else item)

    # Conversion de la colonne arrow
    df_coureur.arrow = df_coureur.arrow.apply(lambda item: float(item[:-1])
                                              if item is not np.nan else item)

    # Suppression des colonnes inutiles
    columns = ['skipper_crew', "nat_sail"]
    df_coureur = df_coureur.drop(columns=columns)
    
    return df_coureur

# Fonction qui nettoie les données scrappées relatives aux caractéristiques des différents bateaux
def clean_df_bateau(df_bateau):  
    # Extraction de la date de lancement
    df_bateau['Date de lancement'] = (df_bateau['Date de lancement']
                              .apply(lambda date: int(re.findall("(\d{4})", date)[0])))

    # Conversion des mesures de longeur, de poids ou de surface en numérique
    columns = ['Longueur', 'Largeur', "Tirant d'eau", 'Hauteur mât',
               'Déplacement (poids)']
    df_bateau[columns] = df_bateau[columns].applymap(length2num)

    columns = ['Surface de voiles au près', 'Surface de voiles au portant']
    df_bateau[columns] = df_bateau[columns].applymap(area2num)

    # Nettoyage de la colonne Nombre de dérives
    df_bateau['Nombre de dérives'] = (df_bateau['Nombre de dérives']
                                      .replace({'2 asymétriques': '2',
                                                'foiler': 'foils'}))

    # Nettoyage de la colonne Voile quille
    df_bateau['Voile quille'] = (df_bateau['Voile quille']
                                      .replace({'Acier mécano soudé': 'acier mécano soudé',
                                                'basculante sur vérin hydraulique': 'basculante avec vérin',
                                                'Inox usiné': 'inox usiné'}))

    # Extraction du numéro de voile et nettoyage du numéro
    df_bateau["Numéro de voile"].replace('16', 'MON 10', inplace=True)
    df_bateau['Numéro de voile'] = df_bateau['Numéro de voile'].fillna('59')

    df_bateau['Numéro de voile'] = (df_bateau['Numéro de voile']
                                    .apply(lambda line: str(line)))
    df_bateau["voile"] = (df_bateau['Numéro de voile']
                          .apply(lambda line: re.findall("(\d+)", line)[0]))
    df_bateau.voile = (df_bateau.voile
                       .replace({"08": "8", "77": "777",
                                 "001": "01", "07": "7", "2": "02"}))

    # Renommage des colonnes
    col2col = {
        'Anciens noms du bateau': 'ancien_noms_bateau',
        'Architecte': 'architecte',
        'Chantier': 'chantier',
        'Date de lancement': 'annee_lacement',
        'Longueur': 'longueur',
        'Largeur': 'largeur',
        "Tirant d'eau": 'tirant_eau',
        'Déplacement (poids)': 'poids',
        'Nombre de dérives': 'derives',
        'Hauteur mât': 'hauteur_mat',
        'Voile quille': 'quille',
        'Surface de voiles au près': 'surface_voiles_pres',
        'Surface de voiles au portant': 'surface_voiles_portant'
    }
    df_bateau = df_bateau.rename(columns=col2col)
    
    # Nouvelles valeurs pour le poids non renseigné de 2 bateaux par la moyenne
    df_bateau.poids = df_bateau.poids.fillna(df_bateau.poids.mean())

    # Suppression des colonnes inutiles
    df_bateau = df_bateau.drop(columns=['Numéro de voile'])
    
    return df_bateau

# Calcul de la zone géographique où sont localisée les bateaux
def generate_zones(df_course):
    zones = np.zeros(len(df_course))
    
    for i in df_course.idx_skipper.unique():
        foo = df_course[df_course.idx_skipper == i].sort_values(by='date_heure')
        idx = foo.geo.apply(lambda point: point[0]>0).argmin()
        if idx == 0:
            zones[foo.geo.index] = 0
            continue
        zone0 = list(foo.geo.iloc[:idx].index)
        zones[zone0] = 0

        foo = foo[idx:]
        idx = foo.geo.apply(lambda point: point[1]>18.4739).argmax()
        if idx == 0:
            zones[foo.geo.index] = 1
            continue
        zone1 = list(foo.geo.iloc[:idx].index)
        zones[zone1] = 1

        foo = foo[idx:]
        idx = foo.geo.apply(lambda point: point[1]>115.1358).argmax()
        if idx == 0:
            zones[foo.geo.index] = 2
            continue
        zone2 = list(foo.geo.iloc[:idx].index)
        zones[zone2] = 2

        foo = foo[idx:]
        idx = foo.geo.apply(lambda point: np.abs(point[1])<67.2892).argmax()
        if idx == 0:
            zones[foo.geo.index] = 3
            continue
        zone3 = list(foo.geo.iloc[:idx].index)
        zones[zone3] = 3

        foo = foo[idx:]
        idx = foo.geo.apply(lambda point: point[0]>0).argmax()
        if idx == 0:
            zones[foo.geo.index] = 4
            continue
        zone4 = list(foo.geo.iloc[:idx].index)
        zones[zone4] = 4


        foo = foo[idx:]
        zone5 = list(foo.index)
        zones[zone5] = 5
        
    df_course['zones'] = zones
    return df_course

📊 Que nous apporte comme informations le découpage en zone ?
En résumé, nous avons la confirmation de ce que nous avions observé auparavant :
  • Atlantique Nord : les voiliers sans foils, avec une vitesse absolue et une vitesse utile plus importante, ont pris l'ascendant sur les foilers.
  • Atlantique Sud : les foilers ont eu des moyennes de vitesse et de distance parcourue beaucoupl plus importante que les autres, rattrapant ainsi leur retard et passant en tête.
  • Océan Indien : les vitesses en moyenne très élevées pour les deux catégories de bateaux reflètent les conditions météos difficiles traversées par les skippers. Grâce à une vitesse utile légèrement supérieure des bateaux sans foils (malgré une vitesse absolue et une distance parcourue moindres, traduisant ainsi des trajectoires plus optimales grâce à une meilleure robustesse et une meilleure navigabilité), ces derniers ont pu rattraper une partie de leur retard.
  • Océan Pacifique : les conditions restent difficiles, et les deux catégories restent au coude à coude. Les bateaux dernière génération ont du mal dans ces conditions à tirer leur épingle du jeu face à des voiliers plus vieux et moins techniques.
  • Océan Atlantique retour : à la faveur d'une météo plus clémente, les foilers prennent définitvement le pas sur les voiliers sans foil et 6 foilers passeront la ligne d'arrivée en moins de 24 heures, contre 2 bateaux classiques.

Conclusion¶

🧭 Le Vendée Globe 2020 / 2021 : "Des formules dans les dunes du Paris-Dakar" ?
La course aura été haletante et de façon surprenante, des voiliers de l'ancienne génération, en moyenne 10 ans plus vieux que les bateaux avec foils, auront résisté jusqu'au bout. Plusieurs observations faites ci-dessus peuvent étayer la remarque faite par Jean Le Cam :
  • Tous les voiliers qui ont abandonné, à l'exception d'un seul, étaient équipés de foils, démontrant la fragilité de cette technologies dans des conditions difficiles, telles que celles traversées durant cette édition. Une étude plus approfondie a montré qu'une largeur de bateau moindre (certainement nécessaire pour l'implantation des foils) pouvait aussi expliqué la fragilité des foilers. Ce dernier point reste à étudier plus en profondeur.
  • La fragilité des foilers a demandé aux skippers plus de prudence dans les mers agitées (l'Océan Indien par exemple), ne pouvant se permettre des trajectoires plus optimales, mais aussi plus exigeantes pour le bateau, contrairement aux voiliers classiques qui ont pu manoeuvrer dans ces eaux de façon plus dynamique et qui ont pu se permettre plus de risques pour augmenter leur vitesse utile pour rester à portée des "formules 1".
Aussi, au vu des analyses menées dans cette étude, la boutade de Jean Le Cam semble tout à fait justifiée, mais reste à nuancer au vu des conditions de navigation de cette édition, particulièrement éprouvantes pour les hommes et leurs bateaux. Toutefois, rien n'indique que le record d'Armel Le Cléac'h soit battu lors de la prochaine édition, à la faveur de vents plus cléments, entérinant par là-même la domination des foilers sur les voiliers à l'ancienne.
